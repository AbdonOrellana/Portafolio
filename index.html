Sobre Mí

Me siento directamente con las jefaturas para entender sus dolores operativos reales. A partir de ahí, diseño y desarrollo plataformas web que eliminan planillas manuales, reducen tiempos de gestión y centralizan información crítica del negocio.

Mi trabajo consiste en reemplazar sistemas legados e ineficientes por soluciones web rápidas, construidas con Django y React. Cuando el proyecto lo requiere, aprendo de forma autónoma herramientas del negocio —como software de ingeniería vial o plataformas GIS— para entregar soluciones que realmente resuelvan el problema de fondo.

Recientemente fui reconocido como Empleado del Mes por mi proactividad al identificar y resolver cuellos de botella operativos antes de que se convirtieran en problemas mayores.

Proyectos Destacados

Sistema de Liquidación y Reportes

Aplicación web full-stack diseñada para reemplazar un sistema legado ineficiente. Optimiza drásticamente la velocidad de procesamiento de datos operacionales y la exportación de reportes críticos para la toma de decisiones.

Django PostgreSQL Pandas Bootstrap Nginx

Características principales:

  • Procesamiento masivo de datos a alta velocidad
  • Generación y exportación automatizada de archivos Excel avanzados
  • Interfaz responsiva con selectores de fecha dinámicos
  • Despliegue robusto en entorno de producción local (On-Premise)
  • Desarrollado en base al levantamiento de requerimientos con jefaturas

Sistema de Gestión de Tickets

Plataforma interna desarrollada para centralizar, priorizar y agilizar el seguimiento de requerimientos. Diseñada bajo un enfoque de mejora continua para facilitar el flujo de trabajo entre los distintos departamentos de la empresa.

Flask Django Python HTML/CSS UI/UX

Características principales:

  • Creación, asignación y seguimiento de requerimientos en tiempo real
  • Interfaz de usuario minimalista, intuitiva y fácil de operar
  • Adaptación de módulos según el feedback directo de los usuarios
  • Gestión de estados y prioridades de resolución
  • Optimización del flujo de comunicación interna
